# ⚡ AthletiQ – Athlete Management System

**AthletiQ** is a premium, high-performance web platform designed for elite coaches and athletes. It unifies training data, performance metrics, and communication into a single, beautifully designed workspace, enabling teams to reach their peak potential through data-driven insights.

---

## 🚀 Key Features

-   **📊 Performance Analytics**: Visualize every rep, sprint, and session with interactive charts and trend analysis.
-   **🏋️ Training Program Builder**: Build periodized plans with drag-and-drop simplicity, customized for each athlete.
-   **🩺 Injury & Recovery Tracker**: Monitor health data to optimize workloads and minimize injury risks.
-   **💬 Coach–Athlete Messaging**: Real-time feedback and video sharing to keep the team aligned.
-   **🖥️ Command Center Dashboard**: A unified view for readiness scores, training loads, and goal progress.

---

## 🛠️ Technical Stack

-   **Frontend**: Professional-grade HTML5 and Vanilla JavaScript.
-   **Styling**: Modern, modular CSS3 architecture using:
    -   **Design Tokens**: Managed in `variables.css` for consistent branding.
    -   **Responsive Layouts**: Optimized for mobile, tablet, and desktop (`responsive.css`).
    -   **Glassmorphism & Micro-animations**: Enhanced user experience with subtle transitions and premium aesthetics.
-   **Structure**:
    -   `assets/css/`: Modular stylesheets (Base, Components, Layout, Home).
    -   `assets/js/`: Core logic and interaction scripts.
    -   `pages/`: Application view templates (Auth, Dashboard).

## 🚦 Getting Started

### Prerequisites

For the current frontend foundation, no complex setup is required. The project is built to be lightweight and fast.

### Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/athlete-management-system.git
    cd athlete-management-system
    ```

2.  **Launch the Application**:
    Simply open the `index.html` file in any modern web browser or use a **Live Server** extension in your IDE (e.g., VS Code) for the best experience.
